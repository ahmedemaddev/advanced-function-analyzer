# Advanced Function Analyzer

A GUI-based mathematical function analyzer built with Python.

## Features
- Function Plotting
- Derivatives Calculation
- Integration
- Critical Points Detection
- Function Transformations
- Interactive GUI

## Technologies Used
- Python
- Tkinter
- SymPy
- NumPy
- Matplotlib

## Project Goal
This project was developed to visualize and analyze mathematical functions using calculus concepts through an interactive graphical user interface.
